## Development

1. Install dependencies:
   ```sh
   $ yarn install
   ```
2. Build source code:
   ```sh
   $ yarn run build
   ```
3. Launch the app:
   ```sh
   $ yarn start
   ```
4. Build installer for this app:
   ```sh
   $ yarn run dist
   ```
